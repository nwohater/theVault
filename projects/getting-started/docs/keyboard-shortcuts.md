# Keyboard Shortcuts

## Global

| Shortcut | Action |
|---|---|
| `Ctrl+T` | Toggle terminal panel |
| `Ctrl+Shift+F` | Toggle search view |
| `Ctrl+S` | Save current document |

## Editor

| Shortcut | Action |
|---|---|
| `Mod+S` | Save document |
| Standard CodeMirror bindings apply | — |

## Sidebar

- **Click** a project folder to expand/collapse
- **Hover** a project to reveal the `+` new document button
- **Right-click** a project or document for rename, delete, and Show in Explorer options

## Terminal

The built-in terminal (`Ctrl+T`) opens a shell in your vault directory. Use it for git operations, running scripts, or anything else you need a command line for.

## Resizing Panels

- Drag the divider between the sidebar and main content to resize
- Drag the divider between editor and preview to adjust the split
- Drag the top edge of the terminal panel to resize it
