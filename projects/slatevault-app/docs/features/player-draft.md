<h1>Player Draft</h1>
<p>The Player Draft feature allows users to participate in a simulated draft, selecting players from a pool of available talent.</p>
<h2>Features</h2>
<h3>Draft Mode</h3>
<p>Players can switch to draft mode, which displays a list of available players with their respective attributes.</p>

<h3>Pick Order</h3>
<p>The pick order is determined by a random draw or a user-specified seeding system.</p>

<h3>Player Selection</h3>
<p Users can select players from the list, and the system will update the team roster accordingly.</p>

<h3>Draft History</h3>
<p>A record of all draft picks and selections is kept for future reference.</p>
